# 💫 About Me:
I am a builder driven by innovation, curiosity, and long-term impact. I believe technology should solve real-world problems rather than exist for the sake of complexity. Every project I create starts with identifying a genuine challenge, researching it thoroughly, and designing a solution that is practical, scalable, and meaningful.<br><br>My interests lie at the intersection of Artificial Intelligence, Finance, Quantitative Research, Data Analytics, and Business Strategy. Rather than limiting myself to a single discipline, I enjoy combining these fields to create intelligent systems that improve decision-making and make complex financial concepts more accessible.<br><br>I approach every project with a product mindset. I don't just write code—I define the problem, analyze users' needs, design the architecture, build the solution, test it, document it, and continuously improve it based on feedback. I believe that great software is not measured by the number of features it has, but by the value it creates for the people who use it.<br><br>Problem solving is the skill I value most. I enjoy breaking down complex challenges into manageable parts, exploring different approaches, and finding efficient, creative solutions. Continuous learning is a core part of who I am, and I actively seek opportunities to expand my knowledge across engineering, finance, AI, entrepreneurship, and emerging technologies.<br><br>I value innovation over imitation. Instead of recreating existing projects, I prefer building products that introduce new ideas, address overlooked problems, or improve existing solutions in meaningful ways. I enjoy experimenting with modern technologies, validating ideas through prototypes, and iterating until they deliver real impact.<br><br>Beyond building products, I believe knowledge should be shared. I enjoy documenting my work, writing technical articles, publishing research, creating demonstrations, and explaining the reasoning behind my decisions so that others can learn from my process. I see open source as an opportunity not only to contribute code but also to contribute ideas, discussions, and educational resources.<br><br>My long-term vision is to build AI-powered products and financial technologies that improve how people interact with information, make decisions, and manage their financial lives. I aspire to lead teams, shape product strategy, and create companies that solve meaningful global challenges through innovation, research, and thoughtful execution.<br><br>On GitHub, every repository represents more than a completed project—it represents a problem explored, a solution engineered, lessons learned, and another step toward building technology that creates lasting value.<br>
